# cartify

A minimal yet powerful frontend e-commerce simulation built with React, TypeScript, and Effector.

This project focuses on modeling cart logic, derived totals, and reactive UI updates using event-driven architecture.

---

##  Overview

Cartify demonstrates how to structure business logic separately from UI while maintaining predictable state management.

It simulates a small online store experience without a backend.

---

##  Features

- Product listing 
- Add to cart
- Increase / decrease quantity
- Remove item
- Real-time total price calculation
- Cart item counter badge
- Persist cart using localStorage

---

##  Technical Highlights

- Effector for state modeling
- Derived store for total price
- Cross-store synchronization
- Feature-based modular structure
- Strict TypeScript typing
